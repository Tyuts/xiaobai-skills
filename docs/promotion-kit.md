## Show HN Body

I built Xiaobai Skills after running into a simple problem: agent skills are
useful, but once you install many of them, broad workflow skills start
overlapping.

This project does two things:

- installs a beginner-friendly default Codex skills stack
- audits installed skills, keeps one global default per need, and moves
  alternatives into a backup folder with restore notes

It is early, Windows/Codex-focused, and intentionally conservative: backup over
deletion, narrow skills over broad global frameworks.

Feedback especially welcome on the default stack and conflict heuristics.

Repo: https://github.com/Tyuts/xiaobai-skills
