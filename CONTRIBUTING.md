# Contributing

Thanks for helping improve Xiaobai Skills.

Good contributions:

- Add safer install or restore workflows.
- Improve conflict detection heuristics.
- Add docs for beginners.
- Report confusing trigger overlaps.
- Suggest high-quality skills for the starter stack.

Please avoid:

- Adding many broad workflow skills as global defaults.
- Removing backup/restore instructions.
- Recommending skills that require unsafe auto-approval without warning.

When suggesting a starter-stack change, explain:

- Which user need it covers.
- Which active skill it replaces or complements.
- Why it should be global instead of backup.
